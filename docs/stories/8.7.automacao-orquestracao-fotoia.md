## Story
**As a** sistema,
**I want** cron jobs e regras de automação para o pipeline FotoIA operar 24/7,
**so that** o ciclo completo de vendas e entrega funcione sem intervenção humana.

## Acceptance Criteria
1. Cron job de follow-ups: verificar leads sem resposta há 24h/48h e acionar agente
2. Orquestrador de pipeline: transições automáticas de status baseadas em eventos
3. Cron job: leads sem resposta após 72h → mover para PERDIDO automaticamente
4. Alertas para o gestor: notificações de pedidos travados, erros de produção, aprovações pendentes
5. Relatórios de performance: relatório semanal automático via email/painel
6. Retry automático em falhas de geração de imagem (max 3 tentativas)
7. Queue para geração de imagens (múltiplos pedidos simultâneos sem timeout)
