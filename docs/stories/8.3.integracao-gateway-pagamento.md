## Story
**As a** sistema,
**I want** integrar com gateway de pagamento (Asaas/Mercado Pago) para gerar cobranças e receber confirmações,
**so that** o fluxo financeiro do FotoIA seja totalmente automatizado.

## Acceptance Criteria
1. Criar cobrança via API do gateway (PIX, boleto, cartão)
2. Endpoint webhook `POST /api/foto-ia/webhook/pagamento` recebe confirmações
3. Webhook valida assinatura HMAC antes de processar
4. Ao confirmar pagamento: `PedidoFotoIA.status` → PAGAMENTO_CONFIRMADO + dispara produção
5. Timeout 48h: reenviar link de pagamento automaticamente
6. Configuração no painel admin: API Key, webhook secret, provedor escolhido
7. Testes em ambiente sandbox do gateway

## Dev Notes
**Asaas API base URL (sandbox):** `https://sandbox.asaas.com/api/v3`
**Asaas API base URL (prod):** `https://api.asaas.com/v3`

**Criar cobrança Asaas:**
```ts
const response = await fetch(`${ASAAS_BASE_URL}/payments`, {
  method: 'POST',
  headers: { 'access_token': ASAAS_API_KEY, 'Content-Type': 'application/json' },
  body: JSON.stringify({
    customer: customerId, // ou criar customer na hora
    billingType: 'PIX',
    value: valorCobrado,
    dueDate: format(addDays(new Date(), 2), 'yyyy-MM-dd'),
    description: `FotoIA - ${tipoFoto}`
  })
})
```
