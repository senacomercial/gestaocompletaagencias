## Story
**As a** admin,
**I want** conectar o WhatsApp da agência via QR Code ou código enviado ao celular,
**so that** a integração fique ativa com o número oficial.

## Acceptance Criteria
1. Página `/whatsapp` acessível apenas por `ADMIN`
2. Estado inicial: botão "Conectar WhatsApp" → emite evento e aguarda QR
3. QR Code exibido como imagem base64 com instruções — atualiza automaticamente se expirar
4. Opção "Conectar via Código": campo de número → código enviado → campo para inserir código
5. Ao conectar: exibe número, foto de perfil e badge "Conectado"
6. Botão "Desconectar" com confirmação
7. Status visível na sidebar (ícone WhatsApp com indicador verde/vermelho)
8. Reconexão automática se sessão salva existir

## Tasks / Subtasks
- [ ] Criar página /whatsapp com guard de ADMIN (AC: 1)
  - [ ] `src/app/(dashboard)/whatsapp/page.tsx`
  - [ ] Verificar `session.user.role === 'ADMIN'` — redirecionar se não for
- [ ] Botão inicial de conexão (AC: 2)
  - [ ] Estado: DESCONECTADO / AGUARDANDO_QR / CONECTANDO / CONECTADO
  - [ ] Botão "Conectar" emite evento Socket.io `whatsapp:connect` com `organizacaoId`
- [ ] Exibir QR Code (AC: 3)
  - [ ] Ouvir evento `qr` do Socket.io
  - [ ] Renderizar QR Code como `<img src={qrBase64} />`
  - [ ] Timer de expiração (30s) com barra de progresso
  - [ ] Ao expirar: reemitir `whatsapp:connect` automaticamente
- [ ] Conexão via código (AC: 4)
  - [ ] Toggle "Conectar via Código" / "Conectar via QR"
  - [ ] Modo código: input de número (formato internacional)
  - [ ] Emitir `whatsapp:pair-code` com número
  - [ ] Exibir código de 8 dígitos recebido + instrução de onde inserir
- [ ] Estado conectado (AC: 5)
  - [ ] Ao receber evento `connected`: mostrar número e foto de perfil
  - [ ] Badge "Conectado" em verde `#22C55E`
- [ ] Botão desconectar (AC: 6)
  - [ ] Diálogo de confirmação
  - [ ] Emitir `whatsapp:disconnect` ao confirmar
- [ ] Indicador na sidebar (AC: 7)
  - [ ] Dot verde (conectado) / vermelho (desconectado) no ícone WhatsApp
  - [ ] Buscar status via API `GET /api/whatsapp/status`
- [ ] Reconexão automática (AC: 8)
  - [ ] Ao carregar página, verificar se existe sessão salva
  - [ ] Se sim: reconectar automaticamente (sem QR)
