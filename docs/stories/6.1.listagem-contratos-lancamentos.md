## Story
**As a** gestor financeiro,
**I want** visualizar todos os contratos e lançamentos de pagamento,
**so that** eu tenha controle completo do fluxo financeiro da agência.

## Acceptance Criteria
1. Página `/financeiro` com tabs: "Contratos", "Lançamentos", "Dashboard"
2. Tab "Contratos": lista com cliente, tipo, VGV, pagamento, recorrência, data, status
3. Expandir contrato: lista de lançamentos vinculados
4. Tab "Lançamentos": filtros por status, mês e cliente
5. ATRASADOS com fundo vermelho sutil; vencendo em 7 dias com borda âmbar
6. Totalizadores: Total Pendente, Total Atrasado, Total Pago no Mês
7. Clicar no cliente navega para `/projetos/[id]`

## Dev Notes
**Depende de:** Story 4.4 (models ContratoFinanceiro, Lancamento)

**API contratos:**
```ts
// GET /api/contratos
const contratos = await prisma.contratoFinanceiro.findMany({
  where: { organizacaoId },
  include: {
    lead: { select: { nome: true, empresa: true } },
    projeto: { select: { id: true } },
    lancamentos: { orderBy: { dataVencimento: 'asc' } }
  }
})
```

**Totalizadores:**
```ts
const pendente = await prisma.lancamento.aggregate({
  where: { status: 'PENDENTE', organizacaoId },
  _sum: { valor: true }
})
```
