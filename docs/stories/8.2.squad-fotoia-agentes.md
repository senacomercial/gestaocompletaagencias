## Story
**As a** sistema,
**I want** os 4 agentes do squad FotoIA implementados e integrados,
**so that** o ciclo de vendas e produção de fotos seja totalmente automatizado.

## Acceptance Criteria
1. Agente Vendedor IA: qualifica lead via WhatsApp, envia proposta, faz follow-up automático
2. Agente Cobrador IA: gera link de pagamento via gateway e monitora confirmação
3. Agente Produtor IA: chama API de geração de imagens e processa revisões
4. Agente Entregador IA: envia galeria ao cliente e coleta aprovação
5. Todos os agentes atualizam `PedidoFotoIA.status` nas transições
6. Logs registrados em `ExecucaoFotoIA` para auditoria
7. Integração com WhatsApp via Baileys (mensagens usando templates do squad)
8. Testes de fluxo completo em ambiente de desenvolvimento

**Padrão de agente:**
```ts
export async function qualificarLead(pedidoId: string) {
  const pedido = await prisma.pedidoFotoIA.findUnique({ where: { id: pedidoId }, include: { lead: true } })
  const template = await loadTemplate('whatsapp-saudacao')
  const mensagem = renderTemplate(template, { nome: pedido.lead.nome })
  await sendWhatsAppMessage(pedido.lead.telefone, mensagem, pedido.organizacaoId)
  await prisma.pedidoFotoIA.update({ where: { id: pedidoId }, data: { status: 'EM_QUALIFICACAO' } })
  await logExecucao(pedidoId, 'qualificar-lead', 'concluido')
}
```
