## Story
**As a** gestor,
**I want** que o Dashboard reflita automaticamente VGV e MRR após cada venda,
**so that** eu tenha visão financeira em tempo real do negócio.

## Acceptance Criteria
1. KPI "VGV Total": soma de `vgvTotal` de leads VENDA_REALIZADA da organização
2. KPI "MRR": soma de `recorrenciaMensal` de contratos RECORRENTE ativos
3. KPI "Taxa de Conversão": leads VENDA_REALIZADA ÷ total de leads do mês
4. Valores atualizados via revalidação após conversão
5. Variação percentual vs mês anterior (seta verde/vermelha)
6. Seção "Últimas Vendas": 5 conversões mais recentes com nome, VGV e data

## Dev Notes
**Queries Prisma:**
```ts
// VGV Total
const vgv = await prisma.lead.aggregate({
  where: { status: 'VENDA_REALIZADA', organizacaoId },
  _sum: { vgvTotal: true }
})

// MRR
const mrr = await prisma.contratoFinanceiro.aggregate({
  where: { tipoPagamento: 'RECORRENTE', projeto: { status: 'ATIVO' }, organizacaoId },
  _sum: { recorrenciaMensal: true }
})
```
