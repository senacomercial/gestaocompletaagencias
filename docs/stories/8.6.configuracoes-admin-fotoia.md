## Story
**As a** gestor,
**I want** um painel de configurações completo para o serviço FotoIA,
**so that** eu personalize preços, prompts de IA e templates de mensagem sem precisar de código.

## Acceptance Criteria
1. `/foto-ia/configuracoes` com tabs: Preços, Prompts IA, Templates WhatsApp, Integrações
2. Tab Preços: editar preço base e quantidade de imagens por tipo
3. Tab Prompts IA: editor de prompt base e negativo por tipo de foto (com preview de variáveis)
4. Tab Templates WhatsApp: editor de mensagem para cada etapa do pipeline
5. Tab Integrações: status das integrações (gateway, Replicate, storage) com teste de conexão
6. Salvar configurações via `PUT /api/foto-ia/configuracoes`
7. Variáveis de template exibidas como chips: `{nome}`, `{empresa}`, `{preco}`, `{link}`
