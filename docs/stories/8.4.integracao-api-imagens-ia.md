## Story
**As a** sistema,
**I want** integrar com Replicate API para gerar imagens profissionais por IA,
**so that** fotos de alta qualidade sejam produzidas automaticamente após pagamento.

## Acceptance Criteria
1. Criar serviço de geração via Replicate API (primary) com fallback DALL-E
2. Endpoint `POST /api/foto-ia/gerar` dispara geração e retorna `predictionId`
3. Webhook de conclusão Replicate salva imagens e atualiza status do pedido
4. Storage de imagens: local (dev) ou Cloudflare R2 (prod)
5. Endpoint `GET /api/foto-ia/imagens/[id]` serve imagem com auth
6. Configuração de prompts editável via painel admin
7. Suporte a múltiplas imagens por pedido (configurável por tipo)

## Dev Notes
**Replicate API:**
```ts
const response = await fetch('https://api.replicate.com/v1/predictions', {
  method: 'POST',
  headers: { 'Authorization': `Token ${REPLICATE_API_TOKEN}`, 'Content-Type': 'application/json' },
  body: JSON.stringify({
    version: REPLICATE_MODEL,
    input: { prompt, negative_prompt, num_outputs: qtdImagens, width: 1024, height: 1024 },
    webhook: `${APP_URL}/api/foto-ia/webhook/replicate`,
    webhook_events_filter: ['completed']
  })
})
const prediction = await response.json()
```
