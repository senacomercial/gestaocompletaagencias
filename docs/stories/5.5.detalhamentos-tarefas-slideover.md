## Story
**As a** membro da equipe,
**I want** abrir uma tarefa e visualizar detalhamentos da gestão,
**so that** eu entenda exatamente o que precisa ser feito sem ambiguidade.

## Acceptance Criteria
1. Clicar no título da tarefa abre slide-over à direita
2. Seção de informações: status, responsável, sprint, prazo (todos editáveis)
3. Seção "Detalhamentos": editor rich text para instruções da gestão
4. Múltiplos detalhamentos em ordem cronológica (feed com autor e data)
5. Botão "Adicionar Detalhamento" com editor inline
6. Histórico de mudanças de status no rodapé
7. Sincronização via Socket.io em tempo real
8. `Ctrl+Enter` salva, `ESC` fecha

## Dev Notes
**Depende de:** Story 5.1 (models), Story 5.4 (TarefaItem)

**Markdown básico:** Usar biblioteca `marked` ou `react-markdown` para renderizar conteúdo dos detalhamentos.

**TarefaHistorico (migration adicional):**
```prisma
model TarefaHistorico {
  id             String       @id @default(cuid())
  tarefaId       String
  usuarioId      String
  statusAnterior StatusTarefa
  statusNovo     StatusTarefa
  createdAt      DateTime     @default(now())
  tarefa         Tarefa       @relation(...)
  usuario        Usuario      @relation(...)
}
```

**Socket.io rooms:** Usar `projeto:{projetoId}` como room para emitir eventos de tarefas.
