## Story
**As a** sistema,
**I want** gerar automaticamente o contrato financeiro e cronograma de pagamentos ao confirmar uma venda,
**so that** o controle financeiro seja iniciado imediatamente com dados corretos.

## Acceptance Criteria
1. `ContratoFinanceiro` criado na mesma transação da Story 4.3
2. Lançamentos gerados conforme tipo: Recorrente (12 mensais), Parcelado (N parcelas), Avulso (1 único)
3. Todos os lançamentos criados com status `PENDENTE`
4. Contrato vinculado ao `Projeto` e ao `Lead`
5. Lançamentos visíveis em `/financeiro` imediatamente
6. Lançamentos com vencimento passado → status `ATRASADO` automaticamente

  - [ ] Criar `ContratoFinanceiro` com projetoId, leadId, organizacaoId, valorTotal, tipoPagamento, dataInicio, recorrenciaMensal, numeroParcelas

  - [ ] Criar função `generateLancamentos(contrato)` → `Lancamento[]`
  - [ ] RECORRENTE: 12 lançamentos mensais, valor = recorrenciaMensal
  - [ ] PARCELADO: N lançamentos, valor = valorTotal/numeroParcelas
  - [ ] AVULSO: 1 lançamento, valor = valorTotal, data = dataVencimento
  - [ ] Todos com `status: PENDENTE`
  - [ ] `tx.lancamento.createMany({ data: lancamentos })`
- [ ] Verificar status ATRASADO (AC: 6)
  - [ ] Ao criar lançamento: se `dataVencimento < new Date()` → `status: ATRASADO`
  - [ ] Cron job diário (ou verificação lazy na query) para atualizar PENDENTE → ATRASADO
  - [ ] API route `GET /api/lancamentos` sempre recalcula status na busca
- [ ] Visibilidade em /financeiro (AC: 5)
  - [ ] Revalidar cache após conversão
  - [ ] Contrato e lançamentos aparecem na aba financeiro

## Dev Notes
**Extensão da transação (Story 4.3 + 4.4 juntos):**
```ts
await prisma.$transaction(async (tx) => {
  // ... lead update + projeto create (Story 4.3) ...

  const contrato = await tx.contratoFinanceiro.create({ data: { projetoId: projeto.id, leadId, organizacaoId, valorTotal, tipoPagamento, dataInicio, recorrenciaMensal, numeroParcelas } })

  const lancamentos = generateLancamentos(contrato)
  await tx.lancamento.createMany({ data: lancamentos.map(l => ({ ...l, contratoId: contrato.id, organizacaoId })) })
})
```

**Verificação de ATRASADO:**
```ts
// Na query GET /api/lancamentos:
const hoje = new Date()
// Atualizar em batch os atrasados:
await prisma.lancamento.updateMany({
  where: { status: 'PENDENTE', dataVencimento: { lt: hoje }, organizacaoId },
  data: { status: 'ATRASADO' }
})
```
